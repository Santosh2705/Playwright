# Playwright Examples

This repository contains example Playwright scripts used for learning and experimentation.

Project structure

- Chapter_01_basics/: basic Playwright/JavaScript examples
- Chapter_02_Java_Functions/: (placeholder)

Quick start

1. Ensure Node.js is installed (v14+ recommended).
2. Initialize and install Playwright test runner:

```bash
npm init -y
npm install -D @playwright/test
npx playwright install
```

3. Run tests (or example scripts):

```bash
npx playwright test
```

Notes

- Files under `Chapter_01_basics/` are JS examples. Update or add tests as needed.
- See `.gitignore` for ignored files.
